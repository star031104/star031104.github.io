Qwen 本地部署记录

最近开始正式折腾本地大模型。

相比直接使用在线 API，本地部署最大的优点其实是：

  • 隐私可控
  • 不受网络限制
  • 响应速度稳定
  • 可以自由改参数
  • 能接各种 Agent 系统

这次主要使用：

进行本地部署。

一、设备环境

目前使用的设备：

  • Honor MagicBook Pro 16
  • RTX 4060 Laptop GPU（8GB）
  • Windows 11
  • WSL2 Ubuntu 22.04

二、为什么选择 Qwen

一开始其实对比过很多模型：

  • DeepSeek
  • Llama
  • Mistral
  • Qwen

最后还是选择了 Qwen。

主要原因：

1. 中文能力强

Qwen 的中文表现确实很好。

尤其：

  • 长文本
  • 中文逻辑
  • 技术问题

明显比很多模型更自然。

2. 本地部署生态成熟

目前：

  • GGUF
  • llama.cpp
  • Ollama

对 Qwen 支持都很好。

部署方便很多。

3. 参数规模适合 4060 Laptop

因为只有 8GB 显存。

所以：

  • 70B 基本不现实
  • 32B 边缘可跑
  • 14B / 7B 更舒服
